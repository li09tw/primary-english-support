# 電子教具遊戲頁面導覽

## 🎯 已建立的遊戲頁面

### 1. 圖像記憶配對 (`/aids/memory-match`) ✅
- **功能**: 記憶配對遊戲，配對英文單字與中文意思
- **需求**: 至少6個單字
- **適合練習**: 單字記憶和視覺聯想
- **遊戲方式**: 
  - 翻開卡片尋找配對的英文單字和中文意思
  - 記住卡片位置，配對成功可繼續翻牌
  - 適合小組活動，訓練記憶力和觀察力
- **特色**:
  - 使用共用TextbookSelector組件
  - 教材和單元選擇
  - 單字預覽
  - 配對統計（移動次數、配對成功數）
  - 遊戲完成提示
  - 強化記憶，適合小組活動，訓練記憶力和觀察力

### 2. 詞彙分類 (`/aids/vocabulary-sort`) ✅
- **功能**: 將單字按類別分類整理
- **需求**: 至少12個單字

  - 6個預定義分類：動物、食物、顏色、數字、家庭、學校

- **特色**:
  - 使用共用TextbookSelector組件
  - 6個預定義分類：動物、食物、顏色、數字、家庭、學校
  - 智能單字分配和分類
  - 拖拽式分類操作
  - 分類統計和完成提示
  - 邏輯思維，分類整理能力，培養邏輯思維

### 3. 「找出不同」單字 (`/aids/find-different`) ✅
- **功能**: 找出不屬於同一類別的單字
- **需求**: 至少15個單字
- **適合練習**: 觀察力和分類識別
- **遊戲方式**: 
  - 在多個單字中找出1個不同類別的單字
  - 訓練觀察力和邏輯推理能力
  - 可個人練習或小組競賽
- **特色**:
  - 使用共用TextbookSelector組件
  - 5個單字中找出1個不同類別的單字
  - 即時答案檢查和解釋
  - 進度追蹤和得分系統
  - 多種分類題型
  - 觀察力訓練，分類識別，提升邏輯推理能力

### 4. 句型拉霸機 (`/aids/sentence-slot`) ✅
- **功能**: 透過拉霸機形式練習句型結構
- **需求**: 至少10個單字
- **適合練習**: 句型創意和語法練習
- **遊戲方式**: 
  - 拉霸機隨機選擇單字填入句型模板
  - 6種句型模板，支援2-3個空格
  - 創造有趣的句子，練習語法結構
- **特色**:
  - 使用共用TextbookSelector組件
  - 6種句型模板，支援2-3個空格
  - 拉霸機隨機選擇單字
  - 即時句子生成和顯示
  - 句型進度和得分追蹤
  - 創意句型，隨機組合練習，創造有趣的句子

### 5. 「是/否」判斷 (`/aids/yes-no-judgment`) ✅
- **功能**: 根據指定分類，快速判斷所呈現的語言元素（單字、詞組或句子）是否符合該分類的定義或陳述
- **需求**: 題庫涵蓋多元詞彙、句型與語法規則，能靈活生成不同難度的題目
- **適合練習**: 
  - 詞性判斷：名詞、動詞、形容詞、副詞等
  - 詞彙分類：動物、食物、交通工具、情緒等
  - 句型判斷：肯定句、否定句、疑問句、祈使句等
  - 語法結構：主詞辨識、動詞時態、助動詞使用、介系詞搭配等
  - 意義理解：邏輯判斷、情境符合度
  - 聽力與閱讀理解：透過聽辨或閱讀快速判斷
- **遊戲方式**: 
  - 遊戲將首先提示一個判斷依據的「分類主題」（例如：「這是名詞嗎？」、「這是一個動物嗎？」、「這個句子是肯定句嗎？」）
  - 接著，會唸出或呈現一個待判斷的「語言元素」（單字、詞組或句子）
  - 玩家需在時間限制內，迅速判斷該語言元素「是」或「否」符合給定的分類主題
  - 例如：
    - 分類主題：「這是動詞嗎？」；呈現單字：「sing」。玩家判斷：「是」
    - 分類主題：「這是一個水果嗎？」；呈現單字：「carrot」。玩家判斷：「否」
    - 分類主題：「這個句子有助動詞嗎？」；呈現句子：「She likes apples.」。玩家判斷：「否」
  - 遊戲旨在訓練玩家的快速反應、精準判斷及對各類語言知識點的即時提取能力
  - 玩法靈活多變，適合個人限時挑戰，亦可作為小組搶答、班級競賽的互動工具，有效提升學習氛圍
- **特色**:
  - 多元題型覆蓋：可切換練習詞性、詞彙分類、主詞、助動詞、句型等豐富判斷類型
  - 使用共用TextbookSelector組件，可依據教材範圍選取題目
  - 智能生成判斷題目，確保題庫豐富且不易重複
  - 即時答案檢查和詳盡解釋，協助玩家理解判斷錯誤的原因
  - 進度追蹤和得分系統，清晰呈現學習成效與進步軌跡
  - 邏輯判斷遊戲，訓練單字理解和邏輯思維

## 🔧 技術實作

### 共用組件系統
- **TextbookSelector**: 統一的教材和單元選擇介面
- **標題更新**: 「選擇教材與單元」→「句型與單字主題」
- **使用 checkbox**: 支援多選教材和單元
- **智能關聯**: 只有選中的教材的單元才可選
- **狀態提示**: 顯示選擇狀態和數量

### 共同功能
- 教材和單元選擇（使用共用組件）
- 單字載入和預覽
- 遊戲狀態管理
- 返回電子教具主頁面
- 響應式設計

### 使用的技術
- Next.js 13+ App Router
- React Hooks (useState, useEffect)
- TypeScript
- Tailwind CSS
