# 電子教具遊戲頁面導覽

## 🎯 已建立的遊戲頁面

### 1. 單字賓果 (`/aids/bingo`) ✅
- **功能**: 5x5賓果網格，隨機叫出單字
- **需求**: 至少25個單字
- **特色**: 
  - 使用共用TextbookSelector組件
  - 教材和單元選擇
  - 單字預覽
  - 賓果檢查（行、列、對角線）
  - 已叫出單字追蹤

### 2. 圖像記憶配對 (`/aids/memory-match`) ✅
- **功能**: 記憶配對遊戲，配對英文單字與中文意思
- **需求**: 至少6個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 教材和單元選擇
  - 單字預覽
  - 配對統計（移動次數、配對成功數）
  - 遊戲完成提示

### 3. 短句接龍 (`/aids/sentence-chain`) ✅
- **功能**: 完成句子模板，練習句型結構
- **需求**: 至少10個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 句子模板填空遊戲
  - 進度追蹤和得分系統
  - 支援跳過和重新開始

### 4. 投影幕問答 (`/aids/projection-qa`) ✅
- **功能**: 大螢幕互動問答遊戲，適合全班參與
- **需求**: 至少8個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 兩種題型：單字意思選擇、英文單字選擇
  - 大螢幕友善的UI設計
  - 即時答案解釋
  - 進度追蹤和得分系統

### 5. 單字拼圖 (`/aids/word-puzzle`) ✅
- **功能**: 拼湊打亂的字母，完成單字
- **需求**: 至少5個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 字母打亂和重組
  - 中文提示輔助
  - 字母使用狀態追蹤
  - 支援刪除和重置操作

### 6. 詞彙分類 (`/aids/vocabulary-sort`) ✅
- **功能**: 將單字按類別分類整理
- **需求**: 至少12個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 6個預定義分類：動物、食物、顏色、數字、家庭、學校
  - 智能單字分配和分類
  - 拖拽式分類操作
  - 分類統計和完成提示

### 7. 「找出不同」單字 (`/aids/find-different`) ✅
- **功能**: 找出不屬於同一類別的單字
- **需求**: 至少15個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 5個單字中找出1個不同類別的單字
  - 即時答案檢查和解釋
  - 進度追蹤和得分系統
  - 多種分類題型

### 8. 句型拉霸機 (`/aids/sentence-slot`) ✅
- **功能**: 透過拉霸機形式練習句型結構
- **需求**: 至少10個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 6種句型模板，支援2-3個空格
  - 拉霸機隨機選擇單字
  - 即時句子生成和顯示
  - 句型進度和得分追蹤

### 9. 黑影圖片猜謎 (`/aids/shadow-guess`) ✅
- **功能**: 從黑影圖片猜出英文單字
- **需求**: 至少5個單字
- **特色**:
  - 使用共用TextbookSelector組件
  - 黑影圖片展示區域
  - 多選項答案選擇
  - 即時答案檢查和解釋
  - 進度追蹤和得分系統

## 🔧 技術實作

### 共用組件系統
- **TextbookSelector**: 統一的教材和單元選擇介面
- **標題更新**: 「選擇教材與單元」→「句型與單字主題」
- **使用 checkbox**: 支援多選教材和單元
- **智能關聯**: 只有選中的教材的單元才可選
- **狀態提示**: 顯示選擇狀態和數量

### 共同功能
- 教材和單元選擇（使用共用組件）
- 單字載入和預覽
- 遊戲狀態管理
- 返回電子教具主頁面
- 響應式設計

### 使用的技術
- Next.js 13+ App Router
- React Hooks (useState, useEffect)
- TypeScript
- Tailwind CSS
- Local Storage 整合

### 共用組件的優勢
- **維護性**: 只需修改一個檔案就能更新所有頁面
- **一致性**: 所有遊戲頁面使用相同的選擇邏輯
- **擴展性**: 新增遊戲頁面時直接使用共用組件
- **用戶體驗**: 統一的選擇介面，降低學習成本
